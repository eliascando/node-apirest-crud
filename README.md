# API RESTful CRUD
Api REST desarrollada en nodejs para efectuuar operaciones CRUD en una base de datos Microsoft SQL Server, utilizando Babel para la transpilacion del codigo JavaScript

## Rutas
#### GET 
##### /products
##### /products/{id}
##### /products/count/all
#### POST
##### /products
#### PUT
##### /products/{id}
#### DELETE
